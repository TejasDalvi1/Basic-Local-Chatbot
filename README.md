# Basic-Local-Chatbot
===============================================================================
## Tech stack:
        (1)Python
        (2)Langchain
        (3)Ollama
        (4)granite3-moe:1b
        (5)Streamlit
-------------------------------------------------------------------------------

## Steps to run:

### (1) Create Environment
```bash
conda create -n localchatbot python=3.10 -y
```
```bash
conda activate localchatbot
```

### (2) Install "requirements.txt"
```bash
pip install -r requirements.txt
```

### (3) Run "app.py":
```bash

streamlit run app.py
```
